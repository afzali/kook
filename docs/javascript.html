اجرای آن توسط جاوا اسکریپت:

$('#myModal').modal(options)

دلخواه ها

نام نوع پیش فرض توضیح
backdrop boolean true افزودن الملن back-drop به زیر پیغام اصلی، بخصوص زمانی که نمی خواهید پنجره شما با کلیک بسته شود.
keyboard boolean true بستن مودال به کمک دکمه escape کیبرد
show boolean true نمایش مودال پس از initialize.

نشانگان

ما می توانیم مودال را بدون جاوا اسکریپت هم اجرا کنیم. کافیست data-toggle="modal" و data-target="#foo" یا href="#foo" را همانند مثال زیر بکار ببرید.

<a class="btn" data-toggle="modal" href="#myModal" >اجرای مودال</a>
                        
<div class="modal" id="myModal">
  <div class="modal-header">
    <a class="close" data-dismiss="modal">×</a>
    <h3>عنوان مودال</h3>
  </div>
  <div class="modal-body">
    <p>متن مودال…</p>
  </div>
  <div class="modal-footer">
    <a href="#" class="btn">بستن</a>
    <a href="#" class="btn btn-primary">ذخیره تغییرات</a>
  </div>
</div>
توجه نمایید! اگر قصد افزودن افکت را دارید کافی است کلاس .fade را به المان .modal خود بفزاید و فایل transition.js را در صفحه صدا بزنید.

دلخواه ها در مودال

برای فعال سازی مودال شما می توانید از دلخواه های مورد نظر خود در قالب object استفاده نمایید.

$('#myModal').modal({
  keyboard: false
})

تغییر دهنده

روش دستی تغییر دهنده در مودال.

$('#myModal').modal('toggle')

نمایش

نمایش دستی مودال.

$('#myModal').modal('show')

پنهان نمودن

پنهان نمودن مودال.

$('#myModal').modal('hide')

رخدادها

رخ دادهای متفاوتی در مودال دیده شده است.

رخ داد توضیح
show این رخداد بلافاصله پس از صدا زدن show صدا زده می شود.
shown زمانی صدا زده می شود که مودال توسط کاربر دیده می شود (البته منتظر می ماند که انیمیشن کامل شود).
hide این رخداد بلافاصله پس از صدا زدن hide صدا زده می شود.
hidden زمانی صدا زده می شود که مودال توسط کاربر دیده می شود (البته منتظر می ماند که انیمیشن کامل شود). 
$('#myModal').on('hidden', function () {
  // do something…
})

پیشنهاد دهنده typeahead.js

در مورد پیشنهاد دهنده

برای پیشنهاد دادن هوشمند ورودی های متنی در فرم ها استفاده می شود.

دانلود فایل

مثال

متن مثالی را وارد نمایید تا با نحوه کار آن آشنا شوید.

نحوه استفاده

به کمک جاوا اسکریپت:

$('.typeahead').typeahead()

دلخواه ها

نام نوع پیش فرض توضیح
source array [ ] اطلاعات اولیه برای نمایش.
items number 8 حداکثر تعداد نمایش پیشنهاد.
matcher function case insensitive متدی برای تعریف نحوه پیدا کردن. که یک ورودی item می گیرد. و در صورت پیدا شدن true بر می گرداند.
sorter function exact match,
case sensitive,
case insensitive		 برای چینش نتایج پیدا شده.که یک ورودی می گیرد به نام items .
highlighter function highlights all default matches برای هایلایت کردن موارد پیدا شده است. که یک ورودی به نام item می گیرد. و حتما باید html برگرداند.

نشانگان

برای این کار مانند نمونه چند خصوصیت اضافه می کنیم.

<input type="text" data-provide="typeahead">
